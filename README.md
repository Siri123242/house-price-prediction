# house-price-prediction
Developed an end-to-end Machine Learning model to predict house prices based on area, number of bedrooms, and bathrooms. Implemented data preprocessing, model training using Linear Regression, and deployed the model using Streamlit for real-time predictions.

#  House Price Prediction

## Project Overview
This project aims to predict house prices based on key features such as area, number of bedrooms, and number of bathrooms using Machine Learning techniques.

##  Objective
To build a complete machine learning pipeline including:
- Data preprocessing
- Feature selection
- Model training
- Model evaluation
- Model deployment

##  Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

##  Model Used
- Linear Regression

##  Workflow
1. Data collection and preprocessing
2. Train-test split
3. Model training
4. Model evaluation using R² score and MSE
5. Model saving using Joblib
6. Deployment using Streamlit

##  Outcome
The model successfully predicts house prices based on user input and is deployed as a web application for real-time predictions.
